## 알고리즘 스터디 (유종원,유기정,최현석)

### 모이는 날 매주 일요일

- 각자 1문제 선택하여 총 3문제 풀어온 뒤, 모임 날짜에 각자 문제풀이 및 설명 의견 공유.
- 파일 경로 > (폴더명:자기이름)/(폴더명:모임날짜)/(파일명:이니셜\_solution\_문제번호\_날짜).java
- 위 와 같은 형식으로 각자 브랜치에 커밋.
- 모임 끝나면 main 브랜치 merge 및 하위 브랜치 merge.
